```js
import express from 'express';
```

**Importation du module Express**

Cette ligne de code est utilisée pour importer le module Express dans votre fichier JavaScript. Express est un framework pour Node.js qui facilite la création d'applications web. Il fournit un ensemble de fonctionnalités pour créer des applications web et mobiles. 


```js
import cors from 'cors';
```

**Importation du module CORS**  

Cette ligne de code signifie que nous importons un module appelé `cors` dans notre application Node.js. `cors` est un package Node.js qui fournit un middleware pour activer le partage de ressources entre origines multiples (Cross Origin Resource Sharing). En d'autres termes, il permet à notre application de communiquer et de partager des ressources avec d'autres domaines.
